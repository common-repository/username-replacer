== Description ==

For livejournal.com, lj.rossia.org, wordpress.org, ya.ru, blogs.mail.ru, lleo.aha.ru users only. &lt;lj user="username"/>, &lt;lj user="username">, &lt;lj comm="community"/>, &lt;lj comm="community">, &lt;ljr user="username">, &lt;ljr comm="community">, &lt;wp user="username">, &lt;ya user="username">, &lt;mr user="username">, &lt;mr comm="community">, &lt;ll user="username">, &lt;hb user="username">, &lt;hb comm="community"> and &lt;bs user="username"> will be replaces with correct HTML code.

== Installation ==

Best way:

Use admin panel for install plugin.

Manual:

1. Upload `username-replacer.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I use it? =

&lt;lj user="username"/>, &lt;lj user="username">, &lt;lj comm="community"/>, &lt;lj comm="community">, &lt;ljr user="username">, &lt;ljr comm="community">, &lt;wp user="username">, &lt;ya user="username">, &lt;mr user="username">, &lt;mr comm="community">, &lt;ll user="username">, &lt;hb user="username">, &lt;hb comm="community"> and &lt;bs user="username"> in edit mode will be replaced with correct HTML code in the posts.
